### Description

SoundFlare is a free and open-source music player built with JS and Electron with customizability in mind.

---
### Prerequisites
- [Node.js](https://nodejs.org/) 
- [Git](https://git-scm.com/)

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/EyelessHairball/SoundFlare.git
   cd soundflare
   ```

2. **Install dependencies:**
   ```sh
   npm install discord-rich-presence electron-squirrel-startup jquery
   ```

3. **Run/build the app:**
   ```sh
   npm start 
   npm build
   ```
